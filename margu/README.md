# Калькулятор ЕГЭ — МарГУ

Часть сайта «Калькулятор ЕГЭ» для Марийского государственного университета.

## Стек

- **Preact** — лёгкий аналог React (3 КБ), меньше boilerplate по сравнению с vanilla
- **Vite** — сборщик, ES6 модули, HMR
- **CSS Modules** — изоляция стилей без лишних зависимостей
- **@preact/signals** — реактивные сигналы вместо useState (меньше перерендеров)
- **preact-router** — лёгкий роутер, маршрут `/calculator`
- **JSDoc** — типизация без TypeScript

## Архитектура (Feature-Sliced Design)

```
src/
├── app/               # Точка входа, роутер
├── pages/
│   └── calculator/    # Страница /calculator
├── widgets/
│   ├── header/        # Шапка сайта
│   ├── footer/        # Подвал сайта
│   ├── score-form/    # Форма ввода баллов
│   └── results-list/  # Список результатов + карточки
├── features/
│   ├── subject-input/ # Поле ввода одного предмета
│   └── filter-results/# Панель фильтрации результатов
├── entities/
│   ├── subject/       # Модель предмета (сигналы, payload)
│   └── speciality/    # Модель специальности (группировка, фильтры)
└── shared/
    ├── api/           # API-клиент + mock-данные
    └── ui/            # Глобальные стили, CSS-переменные
```

## API

Используются два эндпоинта с `https://promo-prk.marsu.ru/promo-prk/web/api/calculator`:

- `GET /subjects` — список предметов ЕГЭ
- `POST /form` — получение подходящих специальностей по баллам

> **Fallback**: Если API недоступен (CORS, сеть), автоматически используются mock-данные из `src/shared/api/mockData.js`.

## Установка и запуск

### Требования
- Node.js 18+ 
- npm 9+

### Шаги

```bash
# 1. Распакуйте архив и перейдите в папку
cd ege-calculator

# 2. Установите зависимости
npm install

# 3. Запустите dev-сервер
npm run dev
```

Приложение откроется на `http://localhost:5173`.

Перейдите по адресу `http://localhost:5173/calculator` для просмотра калькулятора.

### Сборка для продакшена

```bash
npm run build
# Артефакты в папке dist/

npm run preview
# Превью сборки на http://localhost:4173/calculator
```
